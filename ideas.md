# 🎨 Brainstorm Design — Auto GT Concept

## Approche 1 : **Industrial Minimalism** (Probabilité : 0.08)

### Design Movement
Minimalisme industriel inspiré des ateliers modernes et de la mécanique précise. Références : Porsche Design, Audi Corporate, Dyson.

### Core Principles
1. **Précision Fonctionnelle** : Chaque élément a un objectif clair, rien n'est superflu
2. **Contraste Stark** : Noir profond sur blanc pur, pas de gris intermédiaire
3. **Géométrie Nette** : Lignes droites, angles aigus, grilles rigoureuses
4. **Espace Blanc Agressif** : Respiration maximale, sections largement espacées

### Color Philosophy
- **Noir Profond** (#0A0A0A) : Autorité, mécanique, professionnalisme
- **Blanc Pur** (#FFFFFF) : Clarté, transparence, confiance
- **Rouge Vif** (#E63946) : Urgence, action, énergie (CTA uniquement)
- **Gris Acier** (#6B7280) : Détails secondaires, subtilité

**Intention Émotionnelle** : Sérieux, compétence, pas de compromis.

### Layout Paradigm
- **Hero Section** : Image full-width + texte overlay en bas à gauche (asymétrique)
- **Services** : Grid 3 colonnes avec icônes minimalistes, pas de cards
- **Formulaire** : Deux colonnes (gauche : contexte, droite : formulaire)
- **Sections** : Alternance gauche/droite pour asymétrie

### Signature Elements
1. **Ligne Horizontale Rouge** : Séparateur de section (1px, 100% width)
2. **Icônes Géométriques** : Formes simples (cercles, carrés, triangles)
3. **Typography Bold** : Titres en sans-serif lourd (800), corps en 400

### Interaction Philosophy
- **Hover Effects** : Changement de couleur instantané (pas de transition)
- **Clics CTA** : Pulse subtle + transition de couleur
- **Scroll** : Révélation progressive des sections (fade-in)

### Animation
- **Entrance** : Fade-in 0.6s ease-out au scroll
- **CTA Hover** : Scale 1.02 + shadow drop (100ms)
- **Formulaire Focus** : Border rouge vif + glow subtle

### Typography System
- **Display** : Montserrat Bold 800 (titres H1, H2)
- **Body** : Inter Regular 400 (texte corps)
- **Accent** : Montserrat Medium 600 (sous-titres, labels)

## Approche 2 : **Automotive Elegance** (Probabilité : 0.07)

### Design Movement
Luxe automobile moderne avec influences de design de marques premium (Mercedes, BMW). Sophistication et performance.

### Core Principles
1. **Élégance Intemporelle** : Formes courbes et fluides
2. **Hiérarchie Subtile** : Profondeur via ombres et dégradés
3. **Sophistication Typographique** : Polices serif + sans-serif mélangées
4. **Espace Respirant** : Padding généreux, sections larges

### Color Philosophy
- **Noir Charbon** (#1A1A1A) : Fond principal, élégance
- **Blanc Cassé** (#F5F5F5) : Fond secondaire, douceur
- **Bleu Océan** (#0066CC) : CTA, confiance professionnelle
- **Or Discret** (#D4AF37) : Accents premium (badges, icônes)

**Intention Émotionnelle** : Confiance, luxe accessible, professionnalisme.

### Layout Paradigm
- **Hero** : Parallax scroll avec image voiture + texte centré
- **Services** : Cards avec ombre douce, 2 colonnes sur mobile
- **Testimonials** : Carousel avec avatars circulaires
- **Formulaire** : Centré avec fond blanc, shadow enveloppante

### Signature Elements
1. **Dégradé Subtil** : Noir → Gris très léger (background)
2. **Ombres Douces** : Shadow multiple pour profondeur
3. **Icônes Arrondies** : Formes circulaires avec remplissage dégradé

### Interaction Philosophy
- **Smooth Transitions** : 300ms ease-in-out sur tous les hover
- **Micro-interactions** : Petites animations au scroll
- **Feedback Subtil** : Changement de couleur + légère élévation

### Animation
- **Entrance** : Slide-up 0.8s cubic-bezier
- **CTA Hover** : Glow bleu + scale 1.05 (300ms)
- **Cards** : Lift on hover (shadow increase)

### Typography System
- **Display** : Playfair Display Bold 700 (titres H1)
- **Heading** : Montserrat SemiBold 600 (H2, H3)
- **Body** : Lato Regular 400 (texte corps)

## Approche 3 : **Mechanical Brutalism** (Probabilité : 0.06)

### Design Movement
Brutalisme mécanique : formes brutes, matériaux visibles, esthétique d'atelier. Références : Constructivisme russe, design industriel brut.

### Core Principles
1. **Authenticité Brute** : Pas de polish superflu, montrer la mécanique
2. **Texture Visible** : Grain, bruit, imperfections intentionnelles
3. **Couleurs Terreuses** : Rouille, béton, acier oxydé
4. **Typographie Imposante** : Polices lourdes et carrées

### Color Philosophy
- **Gris Béton** (#3D3D3D) : Fond principal
- **Rouille Chaude** (#C84B31) : CTA, énergie mécanique
- **Noir Charbon** (#1A1A1A) : Texte, contraste
- **Blanc Cassé** (#E8E8E8) : Accents, clarté

**Intention Émotionnelle** : Robustesse, authenticité, force brute.

### Layout Paradigm
- **Hero** : Image brute + texte overlay en haut à droite (diagonal)
- **Services** : Grille asymétrique (2-1-2 layout)
- **Sections** : Bords décalés, clip-path pour angles
- **Formulaire** : Bordures épaisses, style "boîte en métal"

### Signature Elements
1. **Bordures Épaisses** : 3px+ borders en noir/rouille
2. **Texture de Grain** : Bruit SVG sur les sections
3. **Formes Géométriques Brutes** : Carrés, rectangles, pas de courbes

### Interaction Philosophy
- **Clics Directs** : Pas de transition, changement immédiat
- **Feedback Tactile** : Sensation de "clic mécanique"
- **Hover Agressif** : Inversion de couleurs

### Animation
- **Entrance** : Zoom-in 0.5s sharp (pas de ease)
- **CTA Hover** : Inversion couleur + border pulse
- **Scroll** : Révélation par clip-path diagonal

### Typography System
- **Display** : Courier Prime Bold 900 (titres monospace)
- **Heading** : IBM Plex Mono SemiBold 600
- **Body** : IBM Plex Mono Regular 400

## ✅ Approche Sélectionnée : **Industrial Minimalism**

**Raison du choix :**
- Parfait pour un garage automobile professionnel
- Contraste stark (noir/blanc/rouge) crée de l'urgence pour les CTA
- Minimalisme = confiance et transparence (valeurs clés de la PRD)
- Géométrie nette = mécanique précise (métier du garage)
- Espace blanc agressif = focus sur les CTAs et formulaires

**Validation Design** : Chaque élément renforce l'objectif business (conversion leads).
